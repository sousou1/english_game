# AdVenture Capitalist / Egg Inc — 成長曲線の実測分解

■ AdVenture Capitalist(Earth)成長曲線の分解 — 全て wiki 実測値(adventure-capitalist.fandom.com: Businesses / Unlocks (Earth) / Managers / Cash Upgrades / Angel Investors)

【1. 施設テーブル(10事業)】
初期コスト / 係数 / サイクル秒 / 収益:
レモネード $3.738 / 1.07 / 0.6s / $1、新聞 $60 / 1.15 / 3s / $60、洗車 $720 / 1.14 / 6s / $540、ピザ $8,640 / 1.13 / 12s / $4,320、ドーナツ $103,680 / 1.12 / 24s / $51,840、エビ船 $1.244M / 1.11 / 96s / $622,080、ホッケー $14.93M / 1.10 / 384s / $7.46M、映画 $179.2M / 1.09 / 1536s / $89.6M、銀行 $2.15B / 1.08 / 6144s / $1.07B、石油 $25.8B / 1.07 / 36,864s(10.2h) / $29.7B。
導出した設計則:
- ティア間の初期コスト比は厳密に「×12」固定(初段のみ×16)。
- ティア間の生産性($/秒)比は ×3〜6(12.0, 4.5, 4.0, 6.0, 3.0, 3.0, 3.0, 3.0, 4.6)。
- 初号機の回収時間(コスト÷$/秒)は 2.2s → 3s → 8s → 24s → 48s → 192s → 768s → 3,072s → 12,288s → 32,056s と、ティアごとに×2〜4で漸増。「新ティア価格 ≒ 自身の収益1〜4サイクル分」かつ「前ティアの×12」。
- 単価成長係数は序盤〜中盤で高く(新聞1.15)、上位ほど低い(1.07)。1.07^100=868倍に対し1.15^100=117万倍 — 序盤施設をわざと頭打ちにし、上位施設を「長く買い続けられる主力」にしている。

【2. マイルストーン倍率(中毒性の核)】
- 各事業: 25/50/100/200/300/400 体で「サイクル速度×2」(計6回=×64)。さらに「全事業が同時に同閾値到達」でもう×2(Capitalistアンロック、最大計12回=×4096。石油10.2h→約9秒)。
- 500体以降は利益乗算に切替: 500〜2000 は100刻みで「×4」(1000と2000は×5)、2250〜5000 は250刻みで×2〜×5、5250以降250刻みで×3、7000で二重アンロック実質×15、10000で×5。
- クロス事業アンロック: 新聞の保有数が他事業を強化(新聞125→レモネード×2、150→洗車×2、…500→エビ船×11、900→石油×11、1000→新聞自身×7,777,777)。「1つの施設を伸ばすと別の施設が跳ねる」相互依存が買い物の迷いと快感を作る。
- 全事業一括(Capitalist): 25〜400で速度×2×6回、500以降100刻みで利益×2(666/777/1111等の遊び番号含む、5000まで)。
- 別系統で現金アップグレード464個(ほぼ全て利益×3、第1波は $250K〜$250B、全事業×3が$1T)。
中毒性の構造: 「あと3個で×2」という近接目標が常に2〜3個視界にあり、×2(速度)→×4(利益)→×15(演出的ジャックポット)とエスカレートする。

【3. マネージャー=自動化の解禁順】
事業順そのまま: $1,000 / $15,000 / $100,000 / $500,000 / $1.2M / $10M / $111.1M / $555.6M / $10B / $100B。
対基礎コスト比は 267× / 250× / 139× / 58× / 11.6× / 8.0× / 7.4× / 3.1× / 4.7× / 3.9× と単調減少 — 最初の自動化は「その施設を約15〜25台買った頃」にやっと届く遠い目標、後半の自動化は施設数台分で即買い。自動化自体が序盤の主要マイルストーン報酬として設計されている(タップ→自動化解禁→上位ティアのタップへ移行、のループ)。

【4. 桁が上がる速度(実測)】
wiki のパラメータ(係数+速度マイルストーンのみ、×3アップグレード/利益アンロック抜き)で貪欲購入シミュレーションを実施した下限値: $1e3=35秒、$1e6=5.0分、$1e8=42分、$1e9=2.3h、$1e10=6.3h、$1e11=26h、$1e12=45.5h。つまり最初の1時間で約8桁、以後は係数だけだと1桁/2〜20時間に急減速する。実ゲームはこの減速を、464個の×3、数百の×2〜×11アンロック、エンジェル(下記)で打ち消し、体感をほぼ一定ペース(初日10桁超、その後も1桁/日前後)に保っている。
- プレステージ(参考): angels = √(生涯収益 ÷ (400B/9))。$44.4Bで1体、$1e15で150体、1体=全利益+2%。生涯収益×1000ごとに角数は×31.6。

■ Egg, Inc. 成長曲線の分解(egg-inc.fandom.com: Eggs / Habitats / Research / Hatchery / Farm Value)

【1. 卵ティア(=ステージ)ラダー】
卵価値: $0.25 → 1.25 → 6.25 → 30 → 150 → 700 → 3,000 → 12,500 → 50,000 → 175,000 → 525,000 → 1.5M → 10M → 1B → 100B → 1T → 15T → 100T。隣接比は ×5 から始まり ×4.7→×4.3→×4→×3.5→×3 と漸減し、後半で×100、×100 の大ジャンプ(プレステージ補正前提)。
解禁条件は農場価値: 1,000万 → 8.4億 → 380億 → 27兆 → … → 5.7e64。隣接ギャップは1.9桁→1.7→2.9→2.5→2.7→3.6→…→4.9桁と漸増。卵19種で約58桁を消化(=1ティア2〜4桁が基準)。

【2. ハブ(容量施設)】
容量 250→500→1,000→2,000→5,000→10,000→…→1億→6億(1-2-5系列、隣接×2〜2.5)。価格は $11.26→$466→$12,266→$340,830→$21.4M→$2.32B→…→3.2e41 で、隣接比×26〜×235(末尾は×3,059、×37,522)。「容量×2に価格×100」という極端な価格付けは、卵価値・産卵率リサーチで収入が指数的に伸びる前提でのみ成立。各レベル内も4段階アップグレード(価格×1.7〜2.6刻み、容量+215%)。

【3. リサーチ(=乗算の供給源)】
1項目 = +5〜25%/Lv × 10〜250Lv、累積で×1.4〜×11/項目(例: Nutritional Supplements +25%×40Lv=×11、Genetic Purification +10%×100Lv=×11)。要所に一発もの: Bigger Eggs ×2(1Lv)、USDE Prime ×3(1Lv)、Even Bigger Eggs ×2×5Lv=×32、Graviton Coating ×2×7Lv=×128、Multiversal Layering ×10×3Lv=×1000。ティアは「前ティアの購入数」で解禁(=安い+10%の連打が次の大物を開ける)。AdCapの「マイルストーン到達ボーナス」をEgg Incは「購入型の細かい階段」に置換している。

【4. タップ経済(コトダマ工房に最も近い)】
タップ=現金でなく「鶏(恒久資本)」を生産。ハッチェリー基礎3羽/秒、容量250(リサーチで最大500)、ホールドで最大33羽/秒。アクティブプレイ係数 Running Chicken Bonus は基礎最大5×、リサーチで +10×/+25×/+300× を積んで最大340×(走っている鶏の数で増減=タップを続けるほど高倍率維持)。放置側は内部孵化器(2〜50羽/分/ハブをリサーチで積層)で、アクティブが常に放置を1〜2桁上回る設計。
【5. 期間実測】最終卵Enlightenmentの最高トロフィーまで50〜78日、序盤3トロフィーで約3日(wiki実測)。ゲーム全体は「週〜月」単位、AdCapは「時間〜日」単位。

## 推奨パラメータ

- **タップ価値の式**: タップ価値 = 覚えた語数 × 職位係数 × アクティブ係数(コンボ)。職位係数は ×2〜×4 のはしご、アクティブ係数は連続正答で1×→5×(将来リサーチで上限拡張)
  - 根拠: Egg Inc: タップ=恒久資本+Running Chicken Bonus(基礎最大5×、リサーチで最大340×)。語彙は線形にしか増えないので指数成分は係数側に置く(Egg Incも鶏数でなくリサーチ乗算が指数源)
- **施設の単価成長率(係数)**: 序盤施設 1.13〜1.15、主力・上位施設 1.07〜1.08(施設ごとに固定)
  - 根拠: AdCap実測: 新聞1.15(100個で価格×117万=意図的な頭打ち)〜石油1.07(100個で×868=長く買える主力)。序盤高・上位低の傾斜が「上位ティアへ移行する理由」を作る
- **ティア間の初期コスト比**: ×10〜12(固定比を推奨)
  - 根拠: AdCap実測: 10事業全てで厳密に×12(初段のみ×16)。Egg Incのハブは×26〜235だが、これは収入の指数成長前提で学習ゲームには過大
- **ティア間の生産性($/秒)比**: ×3〜5(コスト比×12に対して)
  - 根拠: AdCap実測: 12.0, 4.5, 4.0, 6.0, 3.0, 3.0, 3.0, 3.0, 4.6。コスト比>生産性比にすることで上位ティアほど回収時間が×2〜4で伸び、放置価値が出る
- **新ティア初号機の価格**: その施設の収益1〜4サイクル分、かつ前ティア初号機の×12
  - 根拠: AdCap実測: 初号機回収時間 2.2s→3s→8s→24s→48s→192s→768s→3072s→12288s→32056s(コスト÷$/秒)
- **マイルストーン閾値と倍率**: 各施設 25/50/100/200/300/400 で速度(または利益)×2、500以降は100刻みで×4、演出的に7000=×15等のジャックポット。全施設同時到達で追加×2
  - 根拠: AdCap実測のアンロック表そのまま。25/50/100が「常に2〜3個先に見える近接目標」を作る中毒性の核
- **マイルストーン倍率の健全条件**: 倍率 ≤ 係数^25(係数1.07なら×2〜×4が上限、1.07^25=5.43)
  - 根拠: Math of Idle Games(gamedeveloper.com)の解析: 倍率がコスト成長を超えると単一施設連打が永久最適になり経済が壊れる
- **自動化(マネージャー)の価格と解禁順**: 解禁順=施設ティア順。価格はティア1で基礎コストの約250倍(≒その施設15〜25台保有時に届く)から、最上位で3〜8倍まで単調減少
  - 根拠: AdCap実測: $1k/$15k/$100k/$500k/$1.2M/$10M/$111M/$556M/$10B/$100B、対基礎コスト比 267×→3.9×
- **自動化と手動の関係**: 完全代替にしない。放置生産=自動化施設のみ、手動タップにはコンボ係数で常時2〜5×の優位を残す
  - 根拠: Egg Inc: 内部孵化器(放置)に対しアクティブはRunning Chicken Bonusで1〜2桁優位。AdCap型の完全自動化はタップ(=学習)を消す
- **桁が上がる速度の目標**: 初日(初プレイ60分)で6〜8桁、以後1桁/日前後を維持。1ステージ(お題テーマ)=2〜4桁
  - 根拠: AdCapパラメータでの実測シム: 1e6まで5分、1e8まで42分、その後係数だけだと1桁/2〜20hに失速(→乗算アンロックで補う)。Egg Inc実測: 卵1ティア=農場価値1.9〜4.9桁、全19ティアで58桁を50〜78日
- **乗算アンロックの供給密度**: ×2〜×3の購入型アップグレードを常時1〜2個「あと少しで買える」位置に置く(1ティアあたり5〜10個)。語彙系は「同テーマ語を25/50語習得で×2」
  - 根拠: AdCap: 現金アップグレード464個(ほぼ×3)+数百のアンロック。Egg Inc: +10〜25%×多Lvの階段と×2/×3/×1000の一発ものの混合
- **プレステージ(将来枠)**: angels = k × √(生涯灯火)、1体=+2%(加算)。生涯獲得×1000で角数×31.6
  - 根拠: AdCap実測: angels=√(生涯収益÷(400B/9))、$44.4Bで1体、$1e15で150体
- **数字の表示**: 施設容量・目標値は1-2-5系列(250/500/1000/2000/5000…)、通貨は3桁ごとに単位繰り上げ
  - 根拠: Egg Incのハブ容量実測(250→600Mが全て1-2-5系列)。桁インフレ時の可読性を保つ定石

## やってはいけない失敗

1.【係数を全施設一律に高くする】AdCapの新聞は係数1.15で100個買うと価格×1,174,313、レモネード/石油は1.07で×868。1.15は「意図的に引退させる序盤施設」専用の値。全施設1.15にすると中盤で全部が壁になり、全施設1.05にすると買い物の意思決定が消える。施設ごとに1.07〜1.15を傾斜配分するのが実物の答え。

2.【マイルストーン倍率がコスト成長を追い越す】25個ごとに×2は、係数1.07でもコスト側が1.07^25=5.43倍なので安全。これを「25個ごとに×8」等にすると、1つの施設を買い続けるのが永久に最適になり、他施設もマッチングプール(学習)も無意味化する(gamedeveloper.com「The Math of Idle Games」が同条件で警告)。倍率 ≤ 係数^閾値間隔 を必ず守る。

3.【係数とマイルストーンだけで設計を終える】wiki値での実測シム: ×3アップグレード群を抜くと成長は最初の1時間で8桁の後、1桁/2〜20時間まで失速する。AdCapは464個の×3と数百の×2〜×11で、Egg Incは60項目超のリサーチ(累積×6〜×1000)でこの失速を埋めている。「買える乗算」の在庫を切らした時点でプレイヤーは離脱する。

4.【タップ価値の指数成分を語彙数に背負わせる】人間の語彙獲得は線形(1日10〜20語が限界)で、指数インフレのエンジンにはなれない。Egg Incも鶏の数(線形寄り)ではなくリサーチ乗算(×11、×32、×128、×1000)が指数源。コトダマ工房では「覚えた語数」は線形項、「職位・施設・テーマ達成ボーナス」を指数項にしないと、学習が進むほど相対的に報酬が痩せる逆転が起きる。

5.【完全自動化で手動タップを殺す】AdCapはマネージャー10人を揃えた瞬間「眺めるだけのゲーム」になる(実際、中盤以降のAdCapはタップ要素ゼロ)。学習ゲームでこれをやると学習接点が消える。Egg Inc方式 — 放置は内部孵化器でゆっくり、手動はRunning Chicken Bonus(基礎最大5×、拡張で340×)で常に優位 — を採ること。自動化は「タップの代替」ではなく「下位ティアの卒業証書」として、施設15〜25個分相当の価格(基礎コストの約250倍から漸減)で売る。

6.【長い生産サイクルを短縮手段なしで置く】AdCapの石油は初期10.2時間サイクルだが、これは25/50/100/200/300/400の速度半減(個別+全体で最大×4096)で最終9秒まで縮む前提の設計。半減手段を用意せず長サイクルだけ真似ると、ただの放置強制になる。

7.【Egg Incのハブ価格曲線を収入成長の遅いゲームに移植する】容量×2〜2.5に対し価格×26〜235という曲線は、卵価値ラダー(×3〜5/段)とリサーチ乗算が収入を指数で押し上げるから成立する。学習ゲームの収入成長は前述の通り係数依存なので、容量系施設の価格比は×10前後に抑えないと「絶対に買えない施設」が早期に出現する。

8.【プレステージを線形配布にする】AdCapのエンジェルは生涯収益の平方根(√)配布。線形配布にすると2周目以降のインフレが暴走し、平方根より厳しくすると周回動機が消える。将来実装時は √ + 1体2%加算が実証済みの基準点。
