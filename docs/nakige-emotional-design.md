# スペルライト 感動設計パス(泣きゲー文法の適用)

作成: 2026-06-13 / 位置づけ: **提案レイヤー**。確定版 `docs/scenario-structure.md`(v1.0)と現行実装(v6・`handoff-2026-06-13.md`)を上書きしない。レビュー用のたたき台。

根拠(エンタメ分析vault、別リポジトリ `game-design-vault`):
- L2 `knowledge/patterns/earned-catharsis.md` — 喪失駆動型カタルシスの設計術(振幅/モチーフ反復/段階開示/奇跡の正当化/愛着の入口)。**反証レビュー済み**。
- L3 `knowledge/games/genres/nakige.md` — Key/麻枝の系譜と「壊れる条件」(尺不足=愛着不足)。

## 0. 結論(先に)

スペルライトは**素材としては泣きゲーに極めて近い**(喪失=沈黙、奇跡=言葉、世界規則の事前提示、ほろ苦EDの設計思想)。だが現状の最大の弱点は **vault が「壊れる条件」の筆頭に挙げる「尺不足=愛着の蓄積不足」**——218シーンを30日に薄く伸ばす構造は、麻枝のオリジナルアニメ(Angel Beats!/Charlotte)が「1クールに泣きゲーの蓄積を圧縮して失敗した」のと同じ罠に片足を入れている。

**処方は2つ**:
1. **愛着の蓄積を「シーン」でなく「毎日の接触(さしいれ/鐘)」に担わせる**。シーン予算は増やせない。代わりに30日の毎日触る放置ループ自体を“賑やかな日常”の本体にする(§3)。
2. **v1.0とv6で割れている「ノノの位置」を先に決める**。これは感情設計の土台なので、ここが未確定だと以降のすべてが宙に浮く(§2)。

## 1. 現状診断 ― earned-catharsis の5機構で採点

| 機構(L2) | スペルライトの現状 | 評価 |
|---|---|---|
| **①振幅制御**(笑わせて落とす) | レンの軽口・温度3択で「緩」は作れている。だが「急(喪失)」へ落とす設計が章末に散発的で、振幅の頂点(最大の喪失)が**バルドの声喪失=8章**まで来ない | △ 前半の振幅が浅い |
| **②モチーフ反復**(意味変容) | ランタン(言灯)・「おかえり」・木彫りの花飾り・章扉の原文断片と豊富。だが**意味を変えて回帰**させる明示設計が弱い(置いてあるだけ) | ○ 素材は最良。使い方が未設計 |
| **③段階開示**(伏線→一括回収) | 修復率782語・ヴェル真名・5フラグ・大分岐→共通ボス→ルート別ED。**グランドフィナーレ構造は既にある** | ◎ 構造は完成形 |
| **④奇跡の正当化** | 「奪った言葉は腐る」「世界は異界の言葉だけ聞く」を**序盤に事前提示**(L2技法3)。バルドの声喪失=代価(技法2)。隠しED「lonely→friend」=対句反転(Kanon型) | ◎ 教科書的に正しい |
| **⑤愛着の入口** | レンの一人称・ガイ/バルドの掛け合い・選択肢の当事者性。**さしいれ/鐘=毎日の接触**という強力な装置がある | ○ 装置はある。物語的活用が浅い |

**読み取り**: ③④は既にKey作品級に組み上がっている。**伸びしろは①(振幅)と②(モチーフ反復)と⑤(毎日の接触の物語化)に集中**している。逆に言えば、ここだけ手を入れれば「泣ける」濃度は跳ねる。

## 2. 先に決めるべき分岐 ― ノノの位置(感情設計の土台)

**v1.0 と v6 で割れている。これが今いちばん悩ましい点の正体。**

| | v1.0(scenario-structure.md) | v6(handoff-2026-06-13) |
|---|---|---|
| ノノ | 2〜6章は**村に残る**(遠く・不在) | **王都に同行**(ともしび亭住み込み・present) |
| 毎日の接触 | 鐘で届く**手紙** | **さしいれ**(差し入れ) |
| 喪失の型 | 「帰る場所」を離れている寂しさ→6章帰郷で決壊。**故郷型の泣き** | 隣にいる相棒を**失いかける**落差。**Key型(日常を積んで奪う)** |

**推奨: v6(present相棒型)に倒す。** 理由は earned-catharsis §1・§7:
- 喪失の落差は「**失われると最も痛い日常**を前半で作り込む」ほど深くなる。**隣にいて毎日さしいれをくれる相棒**のほうが、遠くの文通相手より「失われると痛い日常」を厚く積める。
- スペルライトの接触構造(30日・毎日触る)は present相棒と相性が良い。手紙(不在の演出)はこの毎日性を活かしきれない。
- ただし**「帰る場所」モチーフは捨てない**。v6では「故郷(村)」でなく「ともしび亭=王都にできた二人の新しい帰る場所」へ付け替える(モチーフの意味変容=§4で再利用)。

⚠️ **v6に倒すなら、scenario-structure.md の §3ノノ欄・5章・6章・幼馴染END「ただいまの呪文」は present 前提で書き直しが要る**。ここを曖昧にしたまま進めると、6章「故郷」章(差分ゼロの共通休憩区間)の前提が崩れる。**この1点だけは実装前にユーザー判断が要る**(本ドキュメントでは推奨を出すに留め、確定はしない)。

## 3. 処方の本丸 ― 「毎日の接触」を賑やかな日常の本体にする

vaultの「壊れる条件」: **尺不足だと愛着が積めず、喪失が「押し付け」に転ぶ**。スペルライトはシーン予算250で固定。シーンを増やせない以上、**愛着の蓄積を放置ループ側(さしいれ・鐘・系統育成)に移す**のが唯一の解。これは麻枝のアニメ失敗を運営(連載)で解いたヘブバンと同じ発想(nakige.md §6)。

具体:
- **さしいれ/手紙を“日常の積み立て”として設計する**。1通ごとは2〜3行でいい。30日×複数通=実シーンより遥かに多い接触回数。ここでノノの口癖・好物・弱音の予兆を**反復**して刷り込む(§4のモチーフ仕込みの主戦場)。
  - 例: 序盤の便りは軽口(「レンレン、また無茶したでしょ」)→中盤に小さな違和感(「……べつに。なんでもない」)→6章で初めての「助けて」。**毎日の便りが伏線の散布装置**になる(earned-catharsis §3a)。
- **鐘(朝昼夜)に感情の時刻を持たせる**。夜の便りだけトーンを一段落とす、等。プレイヤーの生活リズムに同期した「お祭りの賑やかさ→ふとした寂しさ」の微振幅(§1)を、シーンを使わず毎日生む。
- **系統育成(feelings系)とノノの心情をゆるく接続**。feelings語が増えると便りの語彙が少し増える、程度の“皮”。学習進捗が関係性の深まりに見える(ただし**逆は作らない**=下げ機構は禁止。§5の一文ルール)。

> これは「学習の毎日性」をそのまま「日常の積み重ね(愛着)」に転用する手。スペルライト固有の最大の武器で、純ノベルゲーには真似できない。

## 4. 機構別・具体テクニック(章/シーン紐付け)

### ①振幅制御 ― 前半に小さな喪失を分散配置
- 1章 c01_150「戻らない言葉」(老詠唱士の失語は戻らない)は既に正しい“小さな喪失”。**この型を2〜5章の章末にも1つずつ置く**(完全勝利にしない=ほろ苦の予行)。大きな喪失(8章バルド)まで観客の涙腺を“慣らす”。
- 落差の作り方は「笑い⇄泣き」だけでない。**結末先出し(ドラマティック・アイロニー)**も使える(earned-catharsis §1b)。例: ヴェルが序章で「いずれきみは大事なものをひとつ、声と引き換えにする」と零す→以後のバルドの軽口すべてが喪失の予兆に変わる。**裏テーマは言語化しない規則は維持**(予兆は“声と引き換え”止まり、沈黙=恐怖とは言わない)。

### ②モチーフ反復 ― 意味を変えて回帰(最重要の伸びしろ)
スペルライトは素材が最良。**回帰の方向を予測不能にして“あざとさ”を回避**(L2 §2)。
- **ランタン(言灯)**: 序盤=ただの形見/灯し場UI → 終盤=コンプ時に〈星灯〉へ。**中盤に一度“消えかける”描写**を挟むと、終章の再点灯が効く(獲得→喪失→再生)。
- **「おかえり」/「ただいま」**: 幼馴染ENDの最強呪文として既に置かれている。**序盤のさしいれや帰宅シーンで無害に何度も使っておく**→ED で同じ言葉が涙の引き金になる(だんご大家族型の意味変容)。
- **木彫りの花飾り(トバリ村の木工)**: ノノのトレードマーク。**村が静寂の火で焼ける(v6)なら、この飾りが“もう作れない故郷の手仕事”の最後の一個**になる=物の意味が事件で変わる。
- **章扉の原文断片+修復率**: 段階開示の可視化装置として既に優秀。終章で全行が英語で読める=学習の最終目標と物語の最終目標が同一(earned-catharsis §3a の一括回収の最良形)。

### ③段階開示 ― 既に完成。回収漏れだけ点検
- 5フラグ・ヴェル真名・無言王の正体は分割提示済み。**点検: 散布した違和感(ヴェルの声の断片・祖父の過去・修復率)が終章までに全部回収されるか**を実装時にチェックリスト化(投げっぱなしは混乱に転ぶ)。

### ④奇跡の正当化 ― ED の代価を点検
- 隠しED「lonely→friend」は対句反転で◎。**王道ED/闇EDが“代価なき大団円”に見えないか**だけ点検(earned-catharsis §4)。無言王戦の勝利は「最高Stability語=最も根づいた=最も時間をかけた言葉」で決まる設計なので、**勝利の代価=プレイヤーが積んだ30日そのもの**になっている。これは正当化として強い。明文化して残す。

### ⑤愛着の入口 ― レンの声を“共犯”にする
- レンの一人称の軽口は、シリアス転換で**同じ声が黙る**ことで落差を生む(earned-catharsis §5)。**8章バルド声喪失の直後、レンの地の文から軽口が一時的に消える**演出を入れる(説明せず、文体で見せる)。文体規則9(キャラ声固定)の範囲内で“声を失う”を主人公側にも一度だけ反響させる。

## 5. 守るべき一線(泣きゲー化で壊してはいけないもの)

vault が口を酸っぱくする「壊れる条件」と、スペルライト既存規律の一致点。**ここは強化でなく死守**:

1. **学習の罰を感動の喪失に流用しない**(earned-catharsis §8 の明示警告 = スペルライトの一文ルール「誤答は物語を変えない」)。**完全一致。これがあるからスペルライトは“テスト失敗で凹む”泣きゲーにならない。最優先で死守**。誤答=「言葉は胸に戻った」、フラグ下降ゼロ、灯火減らさない、を厳守。
2. **説教禁止**(nakige.md のテンプレ疲れ・押し付け批判)。法則(奪った言葉は腐る等)は敵の描写でのみ。感動の理屈を書いた瞬間に冷める。
3. **裏テーマ(沈黙=声に出せない恐怖)を言語化しない**。泣きゲーの号泣は「説明された感動」では起きない。
4. **音楽の被せで強要しない**(nakige.md の反発要因)。BGMは“幸福の音”を先に置き、喪失後に再生する設計(§2)に留め、悲しい曲を被せて泣かせにいかない。
5. **テスト臭**(終章「詠唱を継ぐ」=穴埋め回避):15歳審査のフォールバック規律を維持。

---

## 6. 実装の優先順(コスト小→効果大)

1. **さしいれ/便りに反復モチーフを仕込む**(§3・§4②)。データ追記だけ。シーン予算ゼロ。効果最大。
2. **2〜5章の章末に“小さな喪失”を1つずつ**(§4①)。既存シーンの差し替えで収まる。
3. **ヴェルの序章予兆(声と引き換え)**(§4①)。1〜2行のログ追記。
4. **ランタン“消えかけ”中盤描写+終章再点灯**(§4②)。
5. **ノノ位置のv1.0/v6確定**(§2)。**←ユーザー判断待ち。これだけ先に要決定**。

## 7. 残リスク

- **30日ペース vs 蓄積**: §3で大幅緩和するが、根治はしない。リリース前に「最初の3日の便りだけで主人公とノノに愛着が湧くか」を15歳ペルソナで観察(nakige.md「愛着の最低蓄積量」の実測)。
- **present相棒に倒すと書き直しコスト**: §2の通り5・6章・幼馴染EDが影響。スコープを見てから決める。
- **モチーフ反復のあざとさ**: 回帰方向を予測不能に(§4②)。15歳審査で「狙いすぎ」と出たら頻度を下げる。
