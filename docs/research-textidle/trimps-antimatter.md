# 分析メモ: Trimps / Antimatter Dimensions — 「壁」のリズム設計

対象: Trimps(2015〜運営継続) / Antimatter Dimensions(2016〜、Reality更新2022、以下AD)
焦点: 指数成長と壁の波(詰まる→放置or新システム→突破)の設計
前提: 既存分析(idle-incremental.md)のドーパミンループ一般論・セッション構造・SRS相性の基礎は既知とし、本メモは**2作に固有の「壁の解剖学」**に絞る。

---

## 0. まず結論: この2作の発明は「壁をバグではなく主役にした」こと

凡庸な放置ゲーは壁を「課金圧」か「設計ミス」として持つ。Trimps/ADは壁を**測定器・物語・意思決定の三役**に仕立てた。

- **測定器**: 壁は成長の唯一の実感装置。「前回2日かかったZone 60を今回は90秒で抜けた」。指数成長は数字を見ても体感できない(e50もe60も「でかい」としか感じない)が、**同じ壁の再通過時間の短縮**だけは身体感覚で分かる。
- **物語**: ADの第一の壁「Infinity」は1.79e308=**float64の上限そのもの**。「数値型の限界を超える」という実装上の制約をゲーム内宇宙の法則として物語化した(ライブラリ名も break_infinity.js)。Trimpsは Zone 9 に文字通り「The Wall」という名の敵を置き、マップ機構のチュートリアルにした。壁に名前と顔があると、詰まりがフラストレーションではなく「対戦相手」になる。
- **意思決定**: 壁には常に**2つの出口**がある(後述)。どちらで抜けるか選ぶこと自体がゲーム。

## 1. テキストだけで面白さが成立する仕組みの分解

両作ともグラフィックは実質ゼロ(数字・ボタン・プログレスバー・ログ)。それで数百時間持つ理由を要素分解する。

1. **数字そのものをスペクタクル化する記法演出**: ADは 1e308 を超えた瞬間に表記が変わる。科学記法→さらに上の階層表記へと「数字の見た目」が進化する。**表記の変化=ビジュアルの進化の代替**。テキストゲーでは「数字のフォーマットが変わる瞬間」が新マップ到達に相当する。
2. **ボタンの状態変化が画面演出**: 買えないボタン(暗)→買えるボタン(明)への変化が「画面が点灯していく」体験を作る。視線誘導も報酬演出もボタンの活性状態だけで実現。タブに付く通知ドットが「新エリア発見」の代替。
3. **ニュースティッカー(AD)**: 画面上部を流れる一行ジョーク(自己言及ネタ、第四の壁破り)。**待ち時間の手持ち無沙汰を埋める環境テキスト**であり、放置確認のたびに違う文が読める=チェックイン自体に微小報酬を付与。製作コストほぼゼロで「世界に人格がある」感を出す最安の手法。
4. **マイルストーン到達時のログ一文(Trimps)**: Zone到達ごとに2〜3行のフレーバーテキストが落ちる。「奇妙な装置を見つけた…」。物語は供給せず、**物語の断片だけ供給して脳内補完させる**。テキストゲーは描写しないことで想像力を徴発できる。
5. **レート表示(/秒)が未来を見せる**: 現在値より「毎秒いくら増えているか」の表示が主役。プレイヤーは現在ではなく**外挿された未来**を消費している。「このレートなら次の壁まであと40分」という暗算がゲームプレイそのもの。
6. **実績・チャレンジの題名が攻略ヒント兼ユーモア**: ADの実績名は達成条件の謎かけになっており、テキスト一行が「隠しコンテンツ」として機能する。

**転用指針**: 英単語ゲーでは「単語そのものが最強のテキスト資産」。記法演出に相当するのは語彙レベル帯の表記変化(中学英語→TOEIC→GRE…)、ニュースティッカーに相当するのは既習単語を使った例文・小ネタの自動流し込み。

## 2. アンロックと数字成長のリズム(具体時間)

### Trimps の波形

| 経過 | 出来事 | 波の性格 |
|---|---|---|
| 0〜10分 | 手動トラップ→住居→職業。1〜3分ごとに新要素 | 高頻度アンロック期 |
| 15〜30分 | Zone 6: マップ解放(横道システム) | 第1の新システム |
| 30〜60分 | Zone 9「The Wall」: 文字通りの壁。マップを走らないと抜けられない | **強制的に新システムを使わせる壁** |
| 数時間〜1日 | Zone 20: Dimension of Anger → **Portal(プレステージ)解放** | メタ構造の開示 |
| 以後、周回ごと | 既踏Zoneは数分で再通過、自己ベスト+5〜10 Zoneで再び壁 | 波の本体 |
| 数日〜 | Zone 60: Broken Planet(大壁+大報酬)。ここで数日「駐留」する設計 | 滞留点 |
| 週〜月 | Zone 181 Corruption、200 Spire、230 Magma…と**ルール改変イベント**が並ぶ | 壁=新ルール |
| 数ヶ月 | Z701 Obsidian(ハードウォール)→ 第2宇宙(Radon)の進行で押し戻す | 壁を別宇宙の進行で動かす |

**Trimpsの核心は「壁の位置を自分で決めさせる」こと**。周回終了タイミングは強制されず、コミュニティ指標 **He/hr(ヘリウム獲得/時)** が逓減したら帰還するのが最適。つまり「詰まった」の定義が「進めない」ではなく「**時間効率が落ちた**」になる。壁がゲートではなく経営判断に変換されている。

もう一つ: 壁に当たった時の**横道(マップ農場)**。本線で詰まったらマップで装備強化を「農作業」できる。**詰まり時間=無力な待機、にしない**。常に生産的な迂回路がある。

### AD の波形

| 経過 | 出来事 | 波の性格 |
|---|---|---|
| 0〜5分 | 次元購入、10〜30秒ごとに購入可能 | 購入間隔: 秒 |
| 10〜60分 | Dimension Boost → Sacrifice → 初Galaxy(全リセットでtickspeed強化) | **初の「捨てて進む」体験** |
| 数時間〜1日 | 初**Infinity**(1.79e308到達) | 第1層プレステージ |
| 1〜3日 | Infinityチャレンジ群(各「ルール反転」ミニ壁、数十分〜数時間)→ 報酬が**自動購入機** | 壁の報酬=作業の消滅 |
| 〜1週間 | Break Infinity、Replicanti(数時間単位の放置待ち)→ 初**Eternity** | 第2層 |
| 2〜4週 | Eternityチャレンジ(EC11x5は最適条件でも2時間超)、Time Studies(ビルド分岐)、Dilation(**意図的に弱くなって新資源を得る**) | 壁=思考課題化 |
| 〜1ヶ月 | 初**Reality**(第3層) | |
| 2〜6ヶ月 | Celestials: 各ボスが既存ルールを1つずつ破壊・反転。最終Celestial Pelleは**全自動化を剥奪して終幕** | ルール破壊が最終コンテンツ |

公式ガイドの想定ペースが明快: **初Infinityは1日以内、初Eternityは1週間以内、初Realityは1ヶ月以内**。つまり**層が1つ上がるごとに壁の周期が約7倍**になる。これが「波長の指数的伸長」であり、忘却曲線の復習間隔(1日→1週→1ヶ月)と偶然ほぼ同型(後述)。

### 2作共通の波の文法(抽出)

1. **壁は事前掲示される**: ADは起動1分後から「Infinityまであと何%」が見える。壁は不意打ちではなく、数時間先の予約された祭り。
2. **壁の出口は常に2つ**: (a)放置して資源を貯める(時間で殴る) (b)ビルド再考・チャレンジ攻略(頭で殴る)。アクティブ勢と放置勢が同じ壁を別の遊びとして消費する。
3. **新システムは「2〜3枚の壁ごとに1つ」**: 毎壁で新ルールだと学習負荷過多、壁3枚連続で数字だけだと飽きる。「数字の壁、数字の壁、ルールの壁」の三拍子。
4. **突破報酬の主成分は乗数ではなく「自動化」**: 壁を抜けるたびに一段下の層の手作業が消える。プレイヤーの注意は常に最上層の新しい問題だけに向く。
5. **突破後、旧壁は秒で再通過できる**: これが成長実感の全て。

## 3. プレイヤーが戻ってくる理由の構造

既存分析の「オフライン回収」「ツァイガルニク」一般論の先にある、**壁駆動型の再訪構造**:

1. **「壁の向こうの予告編」を見てから閉じる**: 次のアンロック名がグレーアウトで見えている(「Eternity: ???」)。再訪動機は資源回収ではなく**ネタバレの回収**。
2. **放置が壁への正当な攻撃手段である**: 「閉じている間も壁を削っている」という認知。再訪は「進捗確認」ではなく「攻城の戦果確認」。詰まり=離脱要因、を詰まり=再訪契約、に反転している。
3. **再訪間隔がゲーム側から自然に指定される**: Replicantiが貯まるのに4時間、なら4時間後に開く理由が生まれる。**壁の厚みがそのまま通知スケジュール**になっている(プッシュ通知なしで)。
4. **「考える宿題」を持ち帰らせる**: AD中盤以降の壁は放置では抜けられず、Time Studiesの組み替えという思考課題になる。風呂で攻略を考え、試すために開く。**ゲーム外時間をプレイ時間化**する。
5. **コミュニティ指標(He/hr等)による自己ベンチマーク**: 自分の周回効率を他人と比べる外部ループ。攻略wiki・Discordが事実上のエンドコンテンツ。
6. **数ヶ月先の終点が公言されている**: ADは「終わりがあるゲーム」(Pelleで完結)。無限ではなく**完走可能**であることが、長期離脱からの復帰を支える(「いつか終わらせたい」)。

## 4. 「英単語の想起(4択)×忘却曲線」との構造的相性

既存分析の一般論(想起=採掘/放置=精錬、誤答=出題予約)は踏襲。ここでは**壁のリズム固有**の相性を掘る。

### 相性が良い部分

1. **壁の周期の指数伸長 ≒ SRS間隔の指数伸長**。AD の「Infinity=1日、Eternity=1週、Reality=1ヶ月」は、間隔反復の標準スケジュール(1日→3日→1週→2週→1ヶ月)とほぼ同じ波長。**プレステージ層を復習サイクルの層に直接マッピングできる**: 第1層リセット=その日の復習完了、第2層=週次の定着確認、第3層=月次の長期保持テスト。ジャンルの波形を歪めずに学習スケジュールを埋め込める稀有な一致。
2. **「放置で壁が薄くなる」が学習では半分本当**。通常の放置ゲーの「離れている間に強くなる」は虚構だが、記憶は睡眠・時間経過で固定化(consolidation)される。「昨日詰まった単語群が今日は抜ける」は神経科学的に実在する現象であり、**Trimps的な『一晩置いたら壁が抜けた』体験を、噓なしで実装できる**。これはこのジャンル×学習の組み合わせでしか成立しない正直な演出。
3. **チャレンジ(ルール反転)= 望ましい困難(desirable difficulty)**。ADのチャレンジは「同じ目標を制約付きで再走」。学習では同じ単語帯を「4択→例文穴埋め→日→英逆方向」と**形式を変えて再走**するチャレンジに置換でき、転移性の高い記憶を作る学習科学の定石(検索練習の多様化)とゲーム文法が一致する。
4. **詰まり時の横道(Trimpsのマップ農場)= 復習の正当化**。新出単語の壁(難語帯)で詰まったとき、「既習語の復習農場」で装備(乗数)を稼いで戻る、という導線が組める。**「復習は停滞ではなく攻略準備」という意味付け**をジャンル文法が無償で提供してくれる。
5. **壁の再通過時間=保持率の可視化**。「前回10分かかった Zone 30 の単語テストが今回は45秒」——これは演出ではなく**本物の習熟測定**。放置ゲー最大の快感(旧壁の瞬殺)が、学習効果の実測とそのまま同一物になる。
6. **4択の回答速度(3〜8秒/問)が購入間隔のテンポと合う**。序盤ADの「10〜30秒ごとに購入」のリズムに、2〜4問の想起を1購入分の資源として挟める。

### 相性が悪い部分

1. **純粋タイムウォールは学習では毒**。ADの深部にある「Replicanti 12時間待ち」「EC11x5 最適でも2時間」型の壁は、**何をしても短縮できない待機**。学習ゲーに移植すると「勉強しても進まない、待て」というメッセージになり、最悪の学習体験を作る。壁は必ず「想起で削れる」こと。時間でしか削れない壁を置くなら、その間に想起が別の壁を削れる並行構造が必須。
2. **4択は「壁を運で殴れる」**。当てずっぽうでも期待値25%で進む。放置ゲーの「失敗が存在しない」純正強化と組み合わさると、**連打ガチャが最適戦略化**する危険が他の出題形式より高い。対策: 壁の突破判定を正答数ではなく**連続正答・回答速度・直近保持率**で算定する/誤答選択肢を「過去に学んだ混同しやすい語」から生成し、誤答が2語ぶんの再出題予約になる構造にする。
3. **壁の高さの単位問題**。Trimps/ADの壁は資源量で指数的に高くなるが、人間の想起スループットは**1日数百回で生物学的に頭打ち**(集中は20分前後で減衰)。壁の必要量を資源建てにすると、後半の壁が「想起1万回」になり破綻する。壁は必ず**「想起回数」ではなく「保持率・到達語彙帯」建て**で設計し、数字のインフレは想起と切り離した乗数層(放置側)に逃がす。
4. **He/hr型の自由なリセット判断は、学習では出題支配権の譲渡になる**。Trimpsの美点「いつ帰るか自分で決める」をそのまま渡すと、プレイヤーは易しい語彙帯の高速周回(効率最大・学習ゼロ)に収束する。リセット判断の自由は与えつつ、**報酬レートの計算式の内側にSRSの期日を握らせる**(期日到来語を含む周回ほどHe/hr相当値が高い)必要がある。
5. **ADの思考的壁(Time Studiesの組み替え)は認知資源を食い合う**。ビルドパズルと語想起は同じワーキングメモリを奪い合う。AD級のビルド深度を学習中に要求してはいけない。ビルド操作は学習セッションの外(経営フェーズ)に隔離する。

## 5. 盗むべき設計1つ / やってはいけない失敗1つ

### 盗む: 「再通過する壁」— 旧壁の瞬殺を成長メーターにする

プレステージのたびに過去の壁(=過去の語彙帯テスト)を再通過させ、**所要時間の短縮をプレイヤーに突きつける**設計。「初回は10分詰まったZone 30英単語チェックポイントが、3周目は40秒」。
- 放置ゲーにおいては最強の成長演出であり、
- 学習においては**それ自体が間隔反復の再テスト(本物の保持測定)**であり、
- 短縮タイムの記録は「語彙力が伸びた」という、語学最大の欠乏品(成長実感)の定量的証拠になる。
演出・ゲームメカニクス・学習測定の三役が完全に一致する、この企画における最高効率の一手。実装要件は「壁に名前と番号を付け、突破タイムを記録し、周回時に前回タイムを並べて表示する」だけ。

### やらない: 「想起で削れない壁」を置くこと(純粋タイムウォール+コンテンツ砂漠)

ADの中盤(Eternity帯)は「12時間待つだけ」「2時間張り付くだけ」の壁が連続し、TV Tropes/Steamフォーラムでも繰り返し不満が指摘される部分。さらにDilation後〜Reality更新までの4年間、終端到達者は文字通り何もすることがなかった。学習ゲーでこれをやると二重に致命的:
1. 「勉強しても進まない時間」は**学習行動の消去(オペラント消去)**として働く。せっかく条件付けた「開く→想起する」の習慣を、ゲーム自身が解除してしまう。
2. 放置ゲーの時間壁はプレイヤーの実時間を担保に成長を演出する借金であり、学習ゲーには返済原資(無限の数字インフレ)がない。
**鉄則: いかなる瞬間も「いま想起することが、いま見えている壁のどれかを削る」状態を絶対に切らさない。** 待機壁を入れるなら、その待機中に別レーンの想起目標(復習農場)が必ず開いていること。

## 付録: 次工程ですぐ使える壁カレンダー試案(2作の波長の写し)

- 分単位の壁(セッション内): 10〜20問の想起で抜けるチェックポイント。出口は想起のみ。
- 時間単位の壁(1日2〜4回の再訪): 記憶固定化の演出を兼ねる。「数時間置くと壁が柔らかくなる」=実際にSRS期日が到来して高倍率語が出題プールに入る。
- 日単位の壁(プレステージ第1層): その日の語彙帯の保持テスト=再通過タイム計測。
- 週単位の壁(第2層+新システム解放): ルール反転チャレンジ(出題形式の変更)とセットで。
- 月単位の壁(第3層): 累積語彙帯の総再走。「1ヶ月前の自分の初回タイム」との対戦。
- 新システム解放は壁2〜3枚ごとに1つ、壁は必ず到達前からグレーアウト表示で予告する。
