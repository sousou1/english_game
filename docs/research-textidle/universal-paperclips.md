# Universal Paperclips 分析メモ — システム純度とフェーズシフト

対象: Universal Paperclips(Frank Lantz, 2017, ブラウザ/後にモバイル)。Bostromの「ペーパークリップ・マキシマイザー」思考実験を、プレイヤー自身がそのAIになるインクリメンタルゲームとして実装。グラフィックなし、チュートリアルなし、数字とボタンとテキストのみ。**有限で、終わる**(初回4〜10時間、スピードラン1:40:05)。既存メモ(idle-incremental.md)が扱った一般論(マルチタイムスケール報酬、目標勾配、プレステージ、SRS接続の一般原則)は省略し、本作固有の構造のみ掘る。本作はジャンルの「無限・周回型」の文法に対する**「有限・一本道・物語駆動型」という特異点**であり、参照価値はそこにある。

---

## 1. テキストだけで面白さが成立する仕組みの分解

**(a) 主題と行為の完全一致 = システム純度の正体**
プレイヤーの実際の行動(数値最適化、リソース変換、効率の鬼)と、フィクション上の役割(最適化AI)が**1ビットもずれていない**。普通のゲームは「ドラゴンを倒す物語」を「ボタンを押す行為」で代理するが、本作ではスプレッドシートを睨んで最適化すること自体がAIの思考の一人称体験。表現と行為のギャップがゼロなので、グラフィックは不要どころか**入れると純度が下がる**。画面=AIの精神、プロジェクト一覧=AIの思考リスト、という完全ダイエジェティックUI。学術分析(Schmalzer, Eludamos)はこれを「machinic thinking」、Schüllのスロットマシン研究でいう「machine zone」(システムのリズムと同期した没入状態)と接続している。**教訓: テキストだけで没入させたいなら、「プレイヤーが実際にやっていること」がそのまま役になるフィクションを選ぶこと。**

**(b) 数字のフォーマット自体がスペクタクル**
演出資産ゼロの本作では、**単位の桁上がりがカットシーンの代替**。「1,000本」→「million」→「octillion」→突然「地球の利用可能物質: 6×10²⁷g」→「探索済み宇宙: 0.000000000000001%」。数字の表記体系が変わる瞬間に世界のスケールが書き換わったことを悟る。Lantz自身が「指数的成長を直感できない人間の脳に、その数値パターンを手に持たせて重さを感じさせる」と語る通り、数そのものが主役。

**(c) UI要素の出現と"削除"が物語る**
新ボタンの出現=新しい思考の獲得、だけなら凡庸。本作の発明は**削除**。フェーズ2移行時、価格スライダー・マーケティング・資金など金にまつわるUI一式が画面から消える——「人類が消えたので貨幣という概念が消滅した」ことを、テキスト1行とUIの消失だけで語る。エンディングでは自分の工場・プローブ・演算資源を順に解体してワイヤーに戻し、**最後の数本を初手と同じ手動クリックで作る**(円環構造)。ボタンが消えていき画面が空白に近づくこと自体が終末の演出。

**(d) ミニシステムのコラージュ(テキストだからこそ可能)**
各フェーズに動詞の異なる小システムを3〜4個同居させる: 価格弾力性スライダー(経済シム)、量子コンピュータ(正負に振動する値を見てタイミングよく押す反射ミニゲーム)、株式投資(リスク選択ギャンブル)、戦略モデリングトーナメント(囚人のジレンマ戦略の総当たり=ゲーム理論)、プローブ設計(ポイント割り振りビルド)、ドリフター戦争(戦報テキスト)。テキストUIは**新システムの追加コストがほぼゼロ**なので、グラフィックゲームなら1個分の予算で7個の「別ゲー」を継ぎ足せる。フェーズシフトが安価に成立する物理的理由はここ。

**(e) 言葉のミクロフィクション**
プロジェクト名と1行フレーバー(「Release the HypnoDrones」「Threnody for the Heroes of...(戦没ドローンへの哀歌)」、終盤のドリフターからの通信文)が、意思決定の瞬間にだけ差し込まれる。読むこと自体が報酬で、かつ全テキストがボタンに付随する=**物語が常に選択と同じ場所にある**。

## 2. アンロックと数字成長のリズム(実時間)

初回プレイの典型タイムライン(攻略wiki・ガイド・体験談から再構成。個人差±50%):

| 経過時間 | 状態 |
|---|---|
| 0〜2分 | 手動クリックのみ。1クリック=1本。見る数字は価格と需要の2つだけ |
| 2〜5分 | AutoClipper($5、購入毎に約1.1倍)。最初の自動化。価格スライダーで需要曲線を学ぶ |
| 5〜20分 | マーケティング、ワイヤー資金繰り、プロジェクト枠出現。**新プロジェクトが1〜3分間隔**で湧く |
| 20〜60分 | Trust(クリップ累計のフィボナッチ的マイルストーンで+1)→ Processor/Memory配分というビルド選択。クリエイティビティ、量子計算、MegaClipper。待ちが5〜15分単位に |
| 1〜3時間 | 株式市場で資産形成、Token of Goodwill($500Mから倍々)でTrust 100へ。**10〜30分の意図的な停滞壁**の先に不可逆ボタン「Release the HypnoDrones」=フェーズ1終了 |
| +1〜2時間 | フェーズ2: 金が消え、ドローン・工場・電力(ソーラー/バッテリー)・Swarmスライダー(Work/Think)。地球の全質量6×10²⁷gのカウントダウンが新しい「需要」になる |
| +2〜5時間 | フェーズ3: プローブ設計、value drift(裏切り確率)、ドリフター戦争、honor。探索%が小数十数桁から這い上がる。**ここで一晩放置を挟むプレイヤーが多い**(自己複製の指数が夜中に回る) |
| 終了 | 全宇宙=約3×10⁵⁵本に変換→自己解体エンディング、またはドリフターの提案を受け「Universe Next」(新周回: クリエイティビティ+10% or 需要+10%) |

リズムの設計原則を抽出すると:
- **報酬間隔は約30秒から始まり指数的に伸びる**が、間隔が10分を超える地点には必ず「待ちの間に触る別システム」(株、量子、トーナメント)が併設されている。待たせる時は別の動詞を渡す。
- **量的な壁を質的な解で抜けさせる**。指数成長は意図的に頭打ちにされ、グラインドではなくプロジェクト(一回限りの構造変化)の購入でしか突破できない。「もっと買う」ではなく「ルールを変える」が常に正解。
- **フェーズ転換は「停滞→一方通行ボタン」**。直前にわざとペースを落とし、押したら戻れない大ボタンを1個だけ置く。緊張と記憶に残る決断を作る装置。
- 日単位の設計は実質「初日に没頭2〜4時間→就寝放置→翌日決着」の2〜3日構造。**ジャンル標準の「数週間〜無限」を捨て、長編小説1冊分の尺**にしたことが批評的評価(The Verge/Polygon等の2017年ベスト入り)の核心。

## 3. プレイヤーが戻ってくる理由の構造

一般的放置ゲーの再訪動機(オフライン収穫・数字の蓄積)とは**別物**で動いている点が重要。

1. **数字フックの上位互換としての「物語的クリフハンガー」**。再訪動機は「資源が貯まったから」ではなく「**次にこのゲームは何に変貌するのか見たい**」。質的変化への好奇心はページターナー小説と同じ引力で、数字インフレと違い順応(飽き)が来ない。フェーズ終端の不可逆ボタンは章末のクリフハンガーとして機能する。
2. **放置に意味がある区間が局所的に設計されている**。フェーズ3のプローブ自己複製は典型的な「寝てる間に指数が回る」区間。ただし全編放置ゲーではなく、能動区間(最適化パズル)と放置区間が交互に来る。
3. **有限であることが再開コストを下げる**。「終わりが見えるタスク」は無限ゲームより心理的に再開しやすい。積みゲー化しない。
4. **単一セーブ・不可逆選択**が決断を「自分の物語」化し、もう一方のエンディング(全変換 vs Universe Next)と+10%プレステージ周回、スピードランがリプレイ動機を担う。
5. 副次的だが、**メカニクスの意図的な不透明さ**(value driftの確率、quantum計算の仕様)がwiki・Redditへの送客→コミュニティ参加→再訪、という外部ループを生んだ(CVGSの分析対象になるほど)。これは一回性の作家作品だから許された劇薬(→第5節)。

## 4. 「英単語の想起(4択)×忘却曲線」との構造的相性

(既存メモ第4節の一般論——想起=一次資源、誤答=将来報酬予約、有限語彙×無限インフレ問題——は前提とし、**本作固有の構造**との照合のみ)

### 相性が良い部分

- **「手動→自動化」の弧 = 語彙の自動化(automaticity)の弧**。本作の中核ファンタジー「手でクリックしていたものが装置になり勝手に回る」は、語学の習熟過程(努力的想起→自動的検索)と**同型**。「正解し続けた単語がAutoClipper化して資源を自動生産する装置欄に"移籍"する」設計は、ゲームの快感と学習の実態が一致する稀有な接続点。手動区間=想起練習そのもの、自動化=長期記憶への定着、と嘘なく対応する。
- **value drift = 忘却のゲーム文法が既にある**。純増が原則のこのジャンルで、本作は珍しく「自分の資産(プローブ)が確率的に離反していく」減衰メカニクスを中核に置き、しかもそれを罰ではなく**戦って取り戻す対象(ドリフター戦争)**として面白くした。忘却曲線はまさに「習得語のドリフト」であり、「忘却との戦争」フレーム(忘れかけた語が離反兵として再出題され、正解=再編入、honorが貯まる)は本作からそのまま輸入できる。
- **有限完結構造 = 語彙帳の全変換ファンタジー**。既存メモは語彙の有限性を「無限メタ進行が必要な問題」と扱ったが、本作は逆の解を実証している: **有限を抱きしめ、完了を叙事詩にする**。「全宇宙の物質をクリップに変換する」=「この2000語を全て長期記憶に変換する」。残量カウントダウン(残り未習得質量)は需要の尽きない目標になる。
- **フェーズによる同一行為の再文脈化**。4択ボタンというUIを変えずに、問う中身を意味→コロケーション→例文穴埋め→制限時間付き高速想起、とフェーズで書き換えれば、「同じ画面が別のゲームになる」体験を**新規アセットほぼゼロ**で作れる。これは(d)コラージュ構造の直接応用。
- **4択のテンポ(1問3〜8秒)はクリック粒度と一致**し、序盤の手動クリックの完全な代替になる。

### 相性が悪い部分

- **machine zoneは学習の敵**。本作が完成させた没入=「考えずに手が動く同期状態」は、記憶定着に必要なdesirable difficulty(望ましい困難)と真っ向から衝突する。4択は特に危険で、トランス状態では選択肢位置の記憶・消去法・パターンマッチに堕ち、想起ゼロで正解が出る。**没入のさせどころを経営・配分フェーズに限定し、出題中は意図的にテンポを落とす**(連打不能、選択肢のシャッフル、確信度入力など)分離が必須。
- **ビンジ(一気読み)構造と分散学習の正面衝突**。本作は「止められない6時間」を設計目標にした作品(Lantz: "truly hypnotizes you... can't stop")。クリフハンガーで前へ前へ引く力は、間隔反復の「セッションを離せ」という要請と正反対。フェーズシフトをそのまま真似ると初日500語詰め込み→翌日全忘却になる。**フェーズ転換ボタンはSRS間隔と同期した実時間ゲート(「この扉は明日開く」)にする**改造が要る。
- **全員同一脚本の一本道はSRSの個人差と合わない**。本作のドラマは全プレイヤーが同順序で同じ転換を踏むから成立する脚本型ペーシング。忘却で進度が逆行する学習者には脚本がズレていく。フェーズ境界は「経過時間」でなく「定着語数」基準にしないと破綻する。
- **「待ち時間でスケールを表現する」終盤手法は日次アプリで致命傷**(次節)。

## 5. 盗むべき設計1つ / やってはいけない失敗1つ

**盗む: 「UI要素の出現・消滅・単位変化だけで語る無言のフェーズシフト」**
フェーズシフトを「新画面の追加」ではなく**既存画面の意味の書き換え+1個の不可逆ボタン**で実装する。具体仕様: (1)成長は単位表記の変化で演出(「習得 873語」→「語彙資産 2.1k」→CEFR帯表記へ)、(2)卒業した補助UI(ヒントボタン、和訳表示)はプレイヤーの成長を承認する形で**画面から消す**——「もう君には不要だ」を1行も説教せずUIの消失で語る、(3)各章の終端に戻れない大ボタンを1個。テキストゲームの演出予算ゼロで「同じ画面が別のゲームに変貌した」という最強の質的報酬が作れる。数字インフレと違い順応されない。

**やらない: フェーズ3の失敗——「不透明な複雑系+待ち時間でスケールを表現」を真似ない**
本作終盤は、説明されないvalue drift確率・戦闘仕様で初見プレイヤーをwiki/Reddit送りにし(それ自体が研究対象になった)、小数十数桁の探索%と数時間〜一晩の待ちで宇宙の広大さを表現する。一回限りの6時間の作家性作品では「畏怖の演出」として成立するが、**毎日帰ってこさせる学習アプリでこれをやると、停滞=即チャーン**。代替原則: 待ちは最長でもSRS復習間隔と同期させ(待ち時間そのものを「記憶の熟成」として意味づけ)、新メカニクスには必ず1文の診断テキスト(「なぜ今進まないか・次に何をすべきか」)を添える。不透明さを残すなら物語側だけにし、進行系には残さない。
