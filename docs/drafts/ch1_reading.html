# c01_002 灯の村の朝

朝。谷あいのナギ村に、今日も鐘が三度鳴った。

ことば一つで湯が沸き、灯ひとつで夜がしのげる。それだけの、小さな村だ。

おれはアキ。字は読めるが、ことばを灯せたためしはない。村の大半がそうだ。

ユイ「アキ、水汲みまだ? 祭りの仕込み、あんたの分も残ってんだけど」

アキ「へいへい。世界はどうしておれにだけ手伝いを増やすかね」

ユイ「世界じゃなくてあたしね。さっさと動く」

演出ノート（c01_002）
演出BGM＝穏やかな朝の旋律（“幸福の音”の地ならし）。SE＝鐘三打・鶏・井戸。立ち絵＝ユイ(normal→smug)。日常パートの開幕＝アキの軽口の一人称（地の文）と、灯火が暮らしを回している村の日常を提示（説明でなく生活の手触りで）。
学習接続なし（日常導入。詠唱・新語供給なし）。
分岐/フラグなし。合流 c01_004。

# c01_004 いつか谷の外へ

井戸の水面に、谷の空がきれいに映っていた。

ユイ「ねえ。あんた、ほんとに村を出る気? ギルドの灯詠みになるって」

アキ「いつかな。字が読めるだけのおれでも、外なら何か変わるかも、ってさ」

ユイ「ふうん。……灯せもしないのに、よく言う」

アキ「うるさい。夢くらい大きいほうが得だろ」

谷の外のことなんて、まだ昔話の挿絵くらいにしか思っていなかった。

――その夜まで、おれは本気でそう信じていた。

演出ノート（c01_004）

学習接続なし。
分岐/フラグなし。合流 c01_006。

# c01_006 祭りの支度

昼下がり、村は祭りの支度で浮き立っていた。提灯を吊るす者、灯を継ぐ者。

老灯守「今年も大灯は満タンだ。爺さんの爺さんの代から、一度も絶やしておらん」

アキ「気の遠くなる話だな。……まあ、絶えないから大灯なんだろ」

ユイ「もう、縁起でもない。……ほら、提灯こっち持って」

何事もなく日が暮れる。村じゅうが、今年いちばんの夜を待っていた。

演出ノート（c01_006）
演出BGM＝祭り前のさざめき（幸福の音を厚く）。SE＝木槌・人の笑い声。立ち絵＝ユイ(smile)。大灯＝「絶えたことのない村の誇り」を老灯守の台詞で立てる（後の消失の落差の土台）。ユイの「縁起でもない」もアイロニー。緩を最大に厚くしてから夜祭→異変へ。
学習接続なし。
分岐/フラグなし。合流 c01_010。

# c01_015 谷の灯

ユイ「そこの提灯、もう消えかけ。世話係をなめないでよね」

ユイは慣れた手で芯を継ぎ、火を移していく。こいつは言葉で灯すのは苦手だ。だから油と芯で繋ぐ。

アキ「おれだって字が読めるだけ。灯せたことなんか一度もない、ふつうの村育ちだよ」

ユイ「灯せる人なんて、谷でも数えるほど。読めるだけで上等でしょ」

ユイ「あたしなんか半分あやしいし。……だから、こうやって油と芯」

谷の縁の大灯だけが、今夜も村じゅうの夜を抱えて、まっすぐ燃えていた。

演出ノート（c01_015）

# c01_020 消える大灯

大灯の火が、谷の端から順に――吸い取られるように消えていった。

祭囃子がやむ。かわりに、柵の外から灰色の靄が這い上がってくる。

老灯守が杖を掲げ、いつものことばを唱える。が、空気が震えるだけで何も灯らない。

老灯守「な……なぜ、ことばが……届かん」

ユイ「アキ。これ、まずいやつ」

演出ノート（c01_020）
演出BGM急停止→無音→低い持続音（翳りのモチーフ初出）。SE＝灯が消える「ふっ」という音の連鎖。画面効果＝周縁から灰色のヴィネット。立ち絵＝ユイ(scared)。振幅: c01_010〜015の幸福の音を断ち切る落差をここで作る。
学習接続なし（世界ルール提示＝「ことばは想起できないと応えない」を住人の失敗で見せる。説教しない）。
分岐/フラグなし。

# c01_025 灯らぬことば

異変を聞きつけ、灯を継ぐ家の者が谷じゅうから集まってきた。灯せる、数少ない者たちだ。

その全員が杖を掲げ、息を合わせ――それでも、火はひとつも戻らない。

老灯守「灯せる者が総がかりで、これか……ことばが、谷から吸われておる」

ユイ「こんなの聞いたことない。あかりが、土地ごと抜かれてってる」

アキ「……さっきまで祭りだったんだぞ。冗談だろ」

# c01_030 カンテラの熱

腰の古いカンテラ――じいちゃんの形見が、急に熱を持った。

蓋の内側に、文字が浮かぶ。祭りの提灯の村の字とも、祖父の写本の字とも違う、見たこともない形。

なのに、なぜか意味が分かる。村の字を読める奴はいても、こんな文字を読める者は一人もいないはずだ。

アキ「なんで……おれ、これが読めるんだ?」

――ふいに、頭の奥で気だるい声がした。

声「……へえ。読めるんだ、きみ」

アキ「は? 誰だ今の」

声「それは、まだ言わない。それより――灯が消えるのはきみの村だけど。退屈してる場合?」

演出ノート（c01_030）
演出SE＝カンテラの「かちり」。声＝立ち絵なし・ログに地の文と別色で紛れ込む（カンテラの声の演出規約）。画面効果＝カンテラの灯だけが暖色で残り、周囲は灰。「村の日常の字（誰でも読める）」と「カンテラの古文字＝灯詞（誰も読めない）」を本文で区別し、声の「読めるんだ」を設定矛盾でなく F1 の入口として効かせる。
学習接続なし（伏線 F1 設置＝カンテラの声。最強カードの前出し）。
分岐/フラグなし。
