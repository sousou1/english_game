# パーティシステム「灯の同行者」設計仕様 v1.0(DQ5式・スペルライトv3統合)

## 0. 設計原則(不変条件への追記)
- P1 学習が火力の主役: パーティ由来の火力補正は全合算で×1.15上限。討伐数kに対して成長しない定数倍 → 武器健全性条件 ln1.5/ln3.39=0.33<1 は不変(定数はレベルシフトのみで勾配に乗らない)
- P2 SRS不干渉: 絆はSRS状態(定着語数)を読むだけ。書き込み・補正・没収は一切なし。絆値は保存せず常時再計算(story.js REVEALと同じ流儀)
- P3 無罰: 仲間効果は加算/軽減のみ。撤退・誤答・放置で絆もパッシブも下がらない
- P4 感情価値で勝負: 数値は控えめに固定し、掛け合い・手紙・加入一枚絵・筆談が本体

## 1. 加入スケジュール(シナリオ structure v1.0 と無矛盾)
加入を2形態に分ける: 【同行】=物理同行・編成スロット可/【文通・後援】=遠隔加入(ロスターに顔は載るが効果は手紙・送金型)。「ノノは物語上6章まで村」問題は、1章末c01_160の手紙袋(百通)をそのまま加入演出に転用した【文通同行】で解決する — 旅立ちで「誰かを連れる」のではなく「声を連れて行く」。

| 章 | 加入イベント | 形態 | 物語上の根拠(既存シナリオの行をそのまま使う) |
|---|---|---|---|

## 2. 編成: 同行2人+寮(DQ5の馬車)
- アクティブ同行は2人まで(レン+2=画面3人)。控えは「ギルドの寮」(馬車相当)
- 寮メンバー: 効果ゼロ・絆は保持・たまにログへ一言(置き去り感の防止)
- 編成UI: バトル画面のランタン横に同行2人のアイコン(絵文字、絆Lv3以上で生成イラストに差替)。タップで編成パネル。入替は無料・即時・戦闘中以外いつでも
- 編成パネル表示: 顔/名前/パッシブ1行/絆ゲージ(好物分野の定着語数 n/閾値)/絆Lvで解錠した回想リスト

## 3. 仲間効果(パッシブ1+介入演出1)
HP・会心・ゴールド・EXP系に寄せ、火力はヴェル(闇)だけに限定する。

| キャラ | 常時パッシブ | 介入演出(トリガー型・低頻度) | 絆Lv5強化 |
|---|---|---|---|
| ノノ(文通,1末〜) | 手紙既読でその日最初の討伐チャンスの開始HP+15% | 鐘で手紙。絆Lvで追伸が増える | 開始HP+20% |
| ノノ(同行,6末〜) | 討伐チャンス中15秒ごとにHP回復=bossAtk(n)×1.0(5秒間隔13発耐えの正規化が約17発へ、生存+30%) | HP30%以下で戦闘中1回だけ即時回復bossAtk×2「あたし特製の処方箋!」 | 回復間隔12秒 |
| ガイ(3〜) | 被ダメ−15%(竜骨の大杖guard25と乗算可、ただし合算下限は被ダメ50%でクランプ) | フィーバー突入時に追撃: 直近平均タップ価値×5のHP削り(1戦闘1回)。※詠唱不能の男の一撃なので魔素・灯火は生まない=B2(攻撃=収入)を汚染しない | 被ダメ−20%、追撃×8 |
| セシリア(後援4末〜) | 討伐ゴールド+10% | 武器屋・装備強化の価格−10%(口利き) | +15%/−15% |
| バルド(後見3〜/同行5〜) | 獲得EXP+20%(B5準拠: EXPはHPと会心にしか乗らないので安全) | ボス開戦時、その戦闘中会心+4%「忘れかけた言葉ほどよく燃える」 | EXP+30% |
| バルド(8章後) | 筆談「笑え」: コンボガード1回/戦闘(短剣traitと重複時は計2回) | 討伐時に筆談ボードで一言(無音演出。BGMだけが鳴る) | ガード2回/戦闘 |

経済検算: (a)火力はヴェル×1.15が天井で定数 → 健全性条件不変。(b)ガイ追撃はタップ価値比例の定数で1戦1回 → 討伐速度+数%相当、収入は不変。(c)セシリア+10%goldは武器価格(直近中ボスHP建て)と同率で成長するため比率一定=インフレしない。(d)回復・軽減・EXPは収入経路に接続しない。

## 4. 絆システム(学習接続)
- 各キャラに好物分野(words.jsのf属性と1:1): ノノ=food/feelings(薬と心)、ガイ=nature/daily(体と飯)、バルド=travel/school(旅と知恵)、セシリア=business/society(契約と法)、ヴェル=分野不問でL4/L5語(上位語の使い道だけくれる女)
- 絆ポイント=好物分野の定着語数(S≥3)をそのまま参照。閾値: Lv1=10/Lv2=25/Lv3=45/Lv4=70/Lv5=100語(2分野合計の理論上限≈195語、本編定着450語×2/8≈112なので終盤にLv5が自然到達するペーシング)
- 報酬: 各Lvで①寮の回想ショートシーン1本(2〜6行、文体規則準拠)②パッシブ微強化(表参照)③Lv3でアイコンが生成イラスト化。火力報酬は置かない
- 章テーマとの共鳴: 6章(nature/food)=ノノ帰郷章でノノ絆が伸びる、7章(society/business)=セシリア章で…と、「今週はこの分野を招こう」導線が「この子と仲良くなろう」と同一の行為になる。既存の系統特化武器(識)とも方向が揃う
- 誤答・放置で絆は絶対に下がらない(lapseで定着数が一時減っても表示Lvは到達済み最高値を保持=ラチェット式。保存はこの最高値1整数のみ)

## 5. ボス掛け合い(演出)
- トリガー5種: 開戦/結界破壊/本体HP50%/討伐/撤退。同行2人から各トリガー最大1行、1.5秒トースト+ログ残留
- 行数予算: 5キャラ×5トリガー×3行=75行+章ボス専用ペア掛け合い(ノノ×ガイ等、組み合わせ10ペア×1行)=計約85行、上限120行で先切り
- 撤退時は必ず無罰の声掛け(バルド「退くのも仕事だ」/ノノ「生きてりゃ薬は効く」)— retreatScarの心理緩和
- 声の固定: ノノ=短文即レス/ガイ=熱血短文/バルド=兄貴口調(8章後は筆談1行)/セシリア=棘敬語/ヴェル=気だるい皮肉。掛け合いはdata化し誤答・成績に一切連動させない
