# スペルライト ―異界の言葉と無言の王―

**異界の言葉(=英語)で詠唱するファンタジーRPG × 忘却曲線。** 忘れかけた言葉ほど、よく燃える。

▶ **プレイ: https://sousou1.github.io/english_game/**

スマホのブラウザで開いて「ホーム画面に追加」。タイピング不要・タップのみ・ノースクロールの1画面。

## 物語

辺境トバリ村の少年レンは、祖父の形見のランタンで「異界の言葉」を読む力に目覚める。
言葉が王家に独占された世界で、冒険者ギルドだけが「確かに思い出せた数」を位にする実力主義の特区。
幼馴染ノノ、お嬢様セシリア、頭の中に住む皮肉な声ヴェル——選択肢で恋愛ルートや闇堕ちに分岐する(全9章・5エンディング)。
**誤答は物語を変えない。選択だけが物語を変える。**

## 遊び方

- **詠唱バトル**: お題(日本語)に合う英単語タイルをタップ=詠唱攻撃。ダメージ=獲得魔素(攻撃=収入なので、負けても何も失わない)
- **敵**: 倒すとゴールド+EXP。5体ごとにボス(結界を削る→「とどめの間合い」に踏み込むと時限戦)。撤退しても無損失+敵に傷痕が残る
- **詠唱ラッシュ**: 25連でゲージ満タン→⚡解放で10秒×3倍(正解で延長・上限18秒)。終了後45秒以内の再点火で×3.25→×4.0と連鎖
- **武器屋**: 討伐10体ごとに入荷。倍率は累積、特性は装備中の1本(ゴールド建て)
- **修行(焚き火)**: FSRS間隔反復の想起。敵もタイマーもいない隔離空間。ここで言葉が根づくほど詠唱が強くなる
- **学習が火力、プレイ量は生存力**: Lv/EXPはHPと会心にだけ効く。攻撃力は語彙(職位×コレクション×武器)が独占

## 学習の仕組み

- FSRS風の間隔反復(`js/srs.js`)。新しい呪文: 書きとめる→3分→鐘(朝昼夜)→卒業
- 職位(見習い→語り部)=記憶安定性。忘れかけ(R≈0.85)の想起が再燃バースト×3.5
- 章ゲートは定着語数(S≥3)建て——確かな想起だけが物語を進める
- レベル(入門〜上級)×分野(8系統=魔法の系統)を設定で選択。収録782語

## 開発

ビルド不要のVanilla JS+PWA。`npm run serve` / `npm test`(51テスト)。

- `js/battle.js` — 敵・ボス・武器・Lv(敵HP=検証済み経済式のリスキン)
- `js/pool.js` — 詠唱プール・コンボ・ラッシュ(インフレ研究準拠)
- `js/scenario.js` — 分岐シナリオ(第1章18シーン+構造書は docs/scenario-structure.md)
- `js/srs.js` + `js/schedule.js` — FSRS+鐘・学習ステップ(無改造の学習核)
- `docs/` — v3統合仕様・経済仕様・インフレ研究・フィール監査
