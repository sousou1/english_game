# コトダマ・リフォージ 〜錆と忘却の塔〜

**忘れかけた単語ほど、よく燃える。** 英単語学習×ハクスラのスマホ向けブラウザゲーム(PWA)。

▶ **プレイ: https://sousou1.github.io/english_game/**

スマホのブラウザで開いて「ホーム画面に追加」するとアプリとして遊べます。タイピング不要(タップ/スワイプのみ)。

## 遊び方

- 言霊(英単語)カードを炉にくべて塔を登る。各ノードの規定燃料を超えたら突破、5ノード+ボス「忘却獣」を制覇すれば勝利。
- 出題の前に2秒の「想起の間」がある。**↑スワイプ=見切り**(自信があるなら宣言して×1.5)、**↓スワイプ=パス**(正直の申告。ペナルティなし)、タップで4択を出す。
- 正解すると燃料を獲得。**忘れかけの単語を思い出した瞬間が最大火力**(再燃バースト ×3.5)。
- ノード突破ごとに鍛冶具(倍率系の装備)を選んでビルドを組む。
- 間違えた言霊は「暴走」してノード終盤に再来する。敗北しても想起の記録はすべて残る。
- 勝利すると**熟成宝箱**(明日開く=分散学習の第1間隔)。
- リスニング問題(🔊)、例文の空所補充、日→英など出題形式は記憶の成長に応じて増える。

## 学習の仕組み

- 裏でFSRS風の間隔反復(SRS)が全カードをスケジューリング。出題はSRSが決め、ゲームはその皮。
- **レアリティ=記憶安定性**(錆鉄→青銅→白銀→黄金→星鋼)。課金でも周回でも上がらない。想起だけが上げる。
- 報酬曲線は忘却確率R≈0.85(忘れかけ)でピークの山型 → 効率プレイ=最適な復習タイミングに一致。
- レベル(入門〜上級の5段階)×分野(日常/ビジネス/旅行など8分野)を設定で選べる。収録 約780語。

## 開発

ビルド不要のVanilla JS(ES Modules)+PWA。

```bash
npm run serve   # http://localhost:8347
npm test        # ロジックの単体・結合テスト
```

- `js/srs.js` — FSRS風スケジューラ(忘却曲線・再燃バースト)
- `js/game.js` — ラン進行(ノード・手札編成・スコア)
- `js/quiz.js` — 出題生成(4形式・ディストラクタ選定)
- `js/ui.js` — 画面・ジェスチャー・演出
- `data/words.js` — 語彙データ(レベル×分野)

進捗は端末のlocalStorageに保存。設定→エクスポートで移行可能。
