# コトダマ工房 〜まっくらな工房と言霊の灯〜

**ことばが消えた世界で、英単語を思い出して火を灯す。** テキスト放置ゲーム×忘却曲線のPWA。

▶ **プレイ: https://sousou1.github.io/english_game/**

スマホのブラウザで開いて「ホーム画面に追加」するとアプリになります。タイピング不要・タップのみ・1回2〜3分。

## このゲームの契約

> **単語を覚える = その言霊が世話なしで働ける期間が伸びる = 放置効率が上がる。この3つは同一の行為。**

- 英単語を思い出すと「言霊」になって工房に住み、灯火(資源)を生む
- 深く記憶に根づくほど(=FSRSのStabilityが伸びるほど)**長くひとりで働ける**: 見習い→職人→親方→長老→語り部
- 放置すると言霊は「うとうと」して生産が落ちるが、その分**熟成マナ**が積もり、起こす(=思い出す)と再燃バーストで一気に弾ける——**放置で力が増す、が忘却曲線の数式上の事実として成立**
- 復習期日は朝・昼・夜の「鐘」に量子化。1日2〜3分×数回のチェックバックがそのまま最適な間隔反復になる
- 「靄を払う」(画面下)はタップ/長押しのクリッカー。生産を増幅するが、**灯火を生むのは想起だけ**(言霊ゼロでタップしても「風だけでは、火は生まれない」)
- 物語はログ一行ずつ静かに進む。あなたが灯した言葉だけが、この世界で発話される

## 学習の仕組み

- FSRS風の間隔反復(`js/srs.js`)が全カードをスケジューリング。タイマーや制限時間は一切ない(選択肢は自分で「ひらく」まで出ない=遅延提示)
- 新しい言葉: 招く(紹介のみ)→3分後→次の鐘→卒業、の学習ステップ
- 「ピンときた」=選択肢を見る前の確信宣言(検証は6択に難化、成功は大きく根づく)
- アンロックと物語はすべて「確かな想起」(期日後の正答)の数で進む——当てずっぽう連打では何も進まない
- レベル(入門〜上級)×分野(暮らし/商い/旅など8種)を設定で選べる。収録 約780語。リスニング(声の燈)対応

## 開発

ビルド不要のVanilla JS(ES Modules)+PWA。

```bash
npm run serve   # http://localhost:8347
npm test        # ユニット・結合テスト
```

- `js/srs.js` — FSRS風スケジューラ(無改造の核)
- `js/schedule.js` — 鐘(朝昼夜)・学習ステップ・確かな想起
- `js/economy.js` — 生産・熟成マナ・貯蔵・施設(決定論precise精算)
- `js/workshop.js` — ゲーム状態機械
- `js/story.js` / `js/story-lines.js` — 段階開示とナラティブ
- `docs/` — v2設計書・テキスト放置ゲー研究・フィール監査

進捗は端末のlocalStorageに保存(v1からの記憶は自動引き継ぎ)。設定→書き出すで移行可能。旧版(塔)は `v0.1-tower` タグ。
